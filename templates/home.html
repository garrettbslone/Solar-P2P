Documentation

PLANET

POST: /register

Params: planetName, password, systemNum
Register the planet to the server

POST: /login

Params: planetName, password
Login the planet to the server

GET: /planetsInSystem

Params: systemNum
Returns list of planet ids and information in a solar system

Sentiment

GET: /sentiments

Params: planetName, systemNum
Get the sentiments that are meant for that user

POST: /sentiment

Params: from, to, sentiment, sentVal, systemNum
Post sentiment to server for other planets to receive.

DELETE: /sentiment

Params: sentimentId
Delete the the sentiment specified
